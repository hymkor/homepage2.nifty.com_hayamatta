本
文書を生成しているスクリプト bougen.rbを公開します。本スクリプトはこんなテキストファイルをごらんのようなHTML ドキュメントに変換します。

もんじょの場合は、生成する際は

    ruby -Sx bougen.rb -icss ../../www/gray.css 原稿ファイル > HTMLファイル
みたいに、css を include するオプションを入れてます。生成物が文書みたいにならないじゃねぇかとか言わないように。なお、使用には Ruby が必要だす。文法は RD を参考にしました。制限としては

  • ((〜))を入れ子にできない
  • ((〜))の中に空行は入れられない(段落をまたげない)
  • あまり汎用的とはいえない(背景色とかね)

などがあります。いったい、誰が使うねん。こんなもん。
